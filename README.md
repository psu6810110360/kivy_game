# 💻 PC builder
เกมจำลองการเป็นเจ้าของร้านประกอบคอมพิวเตอร์ เขียนด้วย Python Kivy

## 👥 สมาชิกในกลุ่ม


## 🚀 ฟีเจอร์หลัก (Features)
* ระบบซื้อขายอุปกรณ์และประกอบคอม
* ระบบตรวจสอบ Socket (Intel/AMD) และ RAM (DDR4/DDR5)
* ระบบคำนวณการใช้ไฟ (Wattage) ป้องกัน Power Overload
* ระบบ Event ประจำวันที่มีผลกับเกม
